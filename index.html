자주 묻는 질문 (FAQ)

회의실 예약은 어떻게 하나요?

+

1. 목록 페이지에서 원하는 날짜를 클릭합니다.

2. 예약 모달에서 회의실을 선택합니다.

3. 시작 시간과 종료 시간을 선택합니다 (10분 단위).

4. 예약자 이름, 참석자 이름, 회의 목적을 입력합니다.

5. 필요시 "줌 계정도 함께 예약하기" 체크박스를 선택합니다.

6. "예약하기" 버튼을 클릭하여 예약을 완료합니다.

예약을 취소하려면 어떻게 하나요?

+

1. "내 예약" 페이지로 이동합니다.

2. 취소하고 싶은 예약의 "취소" 버튼을 클릭합니다.

3. 확인 메시지에서 "확인"을 클릭하면 예약이 취소됩니다.

또는 달력에서 예약된 이벤트를 클릭하여 상세 정보를 확인하고 취소할 수 있습니다.

예약 시간을 변경할 수 있나요?

+

현재는 예약 시간 변경 기능이 제공되지 않습니다. 시간을 변경하려면 기존 예약을 취소하고 새로운 시간으로 다시 예약해주세요.

줌 계정이 궁금해요

+

줌 계정 정보는 맑은소프트 PPM 게시판에서 확인할 수 있습니다.

줌 계정 확인하기 →

회의실과 줌 계정을 함께 예약할 수 있나요?

+

네, 가능합니다. 회의실을 선택하면 "줌 계정도 함께 예약하기" 체크박스가 나타납니다. 이 체크박스를 선택하면 회의실과 줌 계정을 동시에 예약할 수 있습니다.

예약 시간은 몇 분 단위로 선택할 수 있나요?

+

예약 시간은 10분 단위로 선택할 수 있습니다. 시작 시간과 종료 시간 모두 0분, 10분, 20분, 30분, 40분, 50분 중에서 선택할 수 있습니다.

같은 시간에 여러 회의실을 예약할 수 있나요?

+

네, 가능합니다. 각 회의실은 독립적으로 예약되므로, 같은 시간에 서로 다른 회의실을 예약할 수 있습니다. 단, 같은 회의실의 같은 시간대는 중복 예약이 불가능합니다.

예약 후 알림을 받을 수 있나요?

+

현재는 예약 알림 기능이 제공되지 않습니다. 예약 정보는 "내 예약" 페이지에서 확인할 수 있습니다. 향후 이메일 알림 기능이 추가될 예정입니다.
