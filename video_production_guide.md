## 영상 제작에 필요한 것들

### 1. 필수 소프트웨어

#### 무료 옵션
1. **스크린 레코딩**
   - **OBS Studio** (무료, 추천)
     - 다운로드: https://obsproject.com/
     - 화면 녹화 + 오디오 녹음 가능
     - 실시간 편집 기능
   
   - **Windows 기본 도구**
     - Windows + G (게임 바)
     - 화면 녹화 기능 사용

2. **영상 편집**
   - **DaVinci Resolve** (무료, 전문가급)
     - 다운로드: https://www.blackmagicdesign.com/products/davinciresolve
     - 강력한 편집 기능
     - 색보정, 효과 추가 가능
   
   - **Shotcut** (무료, 간단함)
     - 다운로드: https://shotcut.org/
     - 초보자 친화적
     - 기본 편집 기능

   - **Canva** (온라인, 간단한 편집)
     - 웹: https://www.canva.com/
     - 템플릿 사용 가능
     - 간단한 편집에 적합

#### 유료 옵션 (더 전문적)
- **Adobe Premiere Pro** (월 구독)
- **Camtasia** (일회성 구매)
- **Final Cut Pro** (Mac 전용)

### 2. 제작 과정

#### Step 1: 화면 녹화
1. 맑은스페이스 사이트 열기
   - URL: https://malgn-booking.pages.dev
   - 또는 로컬에서 `index.html` 열기

2. OBS Studio 설정
   - 화면 캡처 소스 추가
   - 오디오 입력 설정 (마이크)
   - 해상도: 1920x1080 권장
   - 프레임레이트: 30fps

3. 녹화할 장면들
   - 달력 화면 전체
   - 날짜 클릭 → 예약 모달
   - 회의실 선택
   - 시간 선택
   - 예약 정보 입력
   - 예약 완료
   - 내 예약 목록
   - 모바일 화면 (선택)

#### Step 2: 편집
1. DaVinci Resolve에서
   - 녹화한 영상 불러오기
   - 스토리보드에 따라 클립 자르기
   - 전환 효과 추가 (페이드, 슬라이드)
   - 텍스트 오버레이 추가
   - 배경 음악 추가
   - 효과음 추가

2. 필요한 추가 자료
   - 맑은소프트 로고 이미지
   - 배경 음악 파일
   - 효과음 파일

#### Step 3: 최종 출력
- 해상도: 1920x1080 (Full HD)
- 형식: MP4
- 코덱: H.264
- 프레임레이트: 30fps
