# 회의실 예약 시스템

회의실 예약을 효율적으로 관리할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- ✅ **회의실 목록 조회**: 사용 가능한 회의실 정보 확인
- ✅ **예약하기**: 날짜, 시간, 회의실을 선택하여 예약
- ✅ **예약 조회**: 내 예약 목록 확인 및 필터링
- ✅ **예약 취소**: 예약된 회의실 취소
- ✅ **시간 충돌 방지**: 이미 예약된 시간대는 예약 불가

## 기술 스택

- HTML5
- CSS3 (모던 UI 디자인)
- Vanilla JavaScript (ES6+)
- LocalStorage (데이터 저장)

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 회의실 목록에서 원하는 회의실 선택
3. "예약하기" 버튼 클릭
4. 날짜, 시간, 예약자 정보 입력
5. 예약 완료!
