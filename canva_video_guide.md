# Canva로 맑은스페이스 영상 만들기 가이드

## Step 1: 비디오 프로젝트 시작하기

1. **Canva 로그인**
   - https://www.canva.com/ 접속
   - 계정으로 로그인

2. **새 디자인 만들기**
   - 우측 상단 "디자인 만들기" 클릭
   - 또는 왼쪽 메뉴에서 "비디오" 선택
   - "비디오" 검색 또는 "비디오" 카테고리 선택

3. **비디오 크기 선택**
   - "YouTube 비디오" (1920 x 1080) 추천
   - 또는 "Instagram 스토리" (1080 x 1920) - 세로형
   - 또는 "Facebook 비디오" (1280 x 720)

## Step 2: 화면 녹화 영상 준비하기

### 먼저 화면 녹화 필요!

**Windows 기본 도구 사용**:
1. `Windows + G` 키 누르기
2. "캡처" 버튼 클릭
3. "녹화 시작" 클릭
4. 맑은스페이스 사이트 데모 진행
5. 녹화 중지

**또는 Loom 사용** (더 쉬움):
1. Loom 설치: https://www.loom.com/
2. 녹화 시작
3. 맑은스페이스 사이트 데모
4. 완료 후 MP4 다운로드

## Step 3: Canva에서 영상 편집하기

### 3-1. 녹화한 영상 업로드
1. 왼쪽 메뉴에서 "업로드" 클릭
2. "파일 업로드" 클릭
3. 녹화한 MP4 파일 선택
4. 업로드 완료 대기

### 3-2. 영상을 타임라인에 추가
1. 업로드한 영상을 드래그하여 캔버스에 놓기
2. 자동으로 타임라인에 추가됨

### 3-3. 영상 자르기 및 편집
1. 타임라인에서 영상 클릭
2. 상단 도구바에서:
   - "자르기" - 불필요한 부분 제거
   - "속도" - 재생 속도 조절 (느리게/빠르게)
   - "필터" - 색상 조정
   - "음소거" - 원본 오디오 제거

### 3-4. 텍스트 추가
1. 왼쪽 메뉴에서 "텍스트" 클릭
2. 원하는 텍스트 스타일 선택:
   - "제목 추가"
   - "부제목 추가"
   - "본문 추가"
3. 텍스트를 캔버스에 드래그
4. 텍스트 내용 수정:
   - "회의실 예약, 이제 더 이상 메신저로 공유하지 마세요!"
   - "맑은스페이스로 모든 회의실 예약을 한눈에!"
   - "✨ 실시간 예약 현황 확인"
   - 등등 (스크립트 참고)

5. 텍스트 스타일 조정:
   - 폰트 크기
   - 색상 (보라색 #4f46e5 추천)
   - 위치
   - 애니메이션 효과

### 3-5. 애니메이션 추가
1. 텍스트 또는 요소 선택
2. 상단 "애니메이션" 클릭
3. 효과 선택:
   - "페이드 인"
   - "슬라이드"
   - "팝"
   - 등등

### 3-6. 배경 음악 추가
1. 왼쪽 메뉴에서 "오디오" 클릭
2. "음악" 검색
3. 무료 음악 선택 (밝고 모던한 느낌)
4. 음악을 타임라인에 드래그
5. 음악 길이 조절 (영상 길이에 맞춤)
6. 볼륨 조절 (너무 크지 않게)

### 3-7. 전환 효과 추가
1. 타임라인에서 클립 사이 공간 클릭
2. "전환" 버튼 클릭
3. 효과 선택:
   - "페이드"
   - "디졸브"
   - "슬라이드"
   - 등등

### 3-8. 로고 추가
1. 왼쪽 메뉴에서 "업로드" 클릭
2. 맑은소프트 로고 이미지 업로드 (logo.jpg 사용 가능)
3. 로고를 캔버스에 드래그
4. 크기 및 위치 조정
5. 투명도 조정 (선택사항)

## Step 4: 씬별 구성 (스크립트 참고)

### 씬 1: 오프닝 (0:00-0:10)
- 로고 이미지 추가
- "맑은스페이스" 텍스트 추가
- 페이드 인 애니메이션
- 배경: 단색 또는 그라데이션

### 씬 2: 문제점 제시 (0:10-0:25)
- 메신저 화면 이미지 또는 녹화 영상
- 텍스트 오버레이:
  - "언제 어디서 회의실을 사용할지 메신저로만 공유하다 보니..."
  - "중복 예약, 시간 충돌, 예약 현황 파악 어려움..."

### 씬 3: 솔루션 소개 (0:25-0:50)
- 맑은스페이스 화면 녹화 영상
- 주요 단계마다 텍스트 오버레이:
  - "직관적인 달력에서 날짜를 선택하고"
  - "회의실과 시간을 간편하게 예약하세요"
  - "줌 계정도 함께 예약할 수 있어요!"

### 씬 4: 주요 기능 (0:50-1:10)
- 기능별 화면 녹화 영상
- 각 기능마다 텍스트:
  - "✨ 실시간 예약 현황 확인"
  - "✨ 중복 예약 자동 방지"
  - "✨ 내 예약 목록 한눈에 보기"
  - "✨ 공휴일 자동 표시"

### 씬 5: 모바일 편의성 (1:10-1:25)
- 모바일 화면 녹화 영상
- 텍스트: "모바일에서도 언제 어디서나"

### 씬 6: 마무리 (1:25-1:40)
- 로고 + 주요 화면 모음
- URL 텍스트: "malgn-booking.pages.dev"
- "이제 메신저 대신 맑은스페이스를 사용하세요!"

## Step 5: 미리보기 및 수정

1. **재생 버튼** 클릭하여 전체 영상 확인
2. 각 씬의 타이밍 조정
3. 텍스트 가독성 확인
4. 음악 볼륨 확인
5. 전환 효과 확인

## Step 6: 다운로드

1. 우측 상단 **"다운로드"** 버튼 클릭
2. 파일 형식 선택:
   - **MP4 비디오** (추천)
   - 해상도: 1080p (Full HD)
3. "다운로드" 클릭
4. 완료!

## 팁과 트릭

### 텍스트 가독성
- 배경이 복잡하면 텍스트 뒤에 반투명 박스 추가
- 폰트 크기는 최소 24px 이상
- 색상 대비 확보 (밝은 배경 → 어두운 텍스트)

### 타이밍
- 각 씬은 최소 3-5초 이상
- 텍스트는 읽을 시간 충분히 확보
- 전환 효과는 0.5-1초

### 음악
- 배경 음악은 30-40% 볼륨
- 텍스트가 나올 때는 음악 볼륨 낮추기
- 영상 끝에서 페이드 아웃

### 색상
- 맑은소프트 브랜드 컬러 활용:
  - 보라색: #4f46e5
  - 빨간색: #ef4444 (공휴일)
  - 파란색: #2563eb (예약)

## 문제 해결

### 영상이 너무 길어요
- 불필요한 부분 자르기
- 재생 속도 빠르게 (1.2x - 1.5x)
- 텍스트 표시 시간 단축

### 텍스트가 안 보여요
- 텍스트 뒤에 반투명 박스 추가
- 폰트 크기 키우기
- 색상 대비 높이기

### 음악이 너무 커요
- 타임라인에서 음악 클릭
- 볼륨 조절 슬라이더 사용
- 30-40% 정도로 설정

### 다운로드가 안 돼요
- Canva Pro 필요할 수 있음 (일부 기능)
- 무료 버전에서는 워터마크 있을 수 있음
- 브라우저 새로고침 후 재시도
