# 맑은스페이스 회의실 예약 시스템 영상 스토리보드

## 전체 구성
- **총 길이**: 1분 40초
- **스타일**: 모던하고 깔끔한 UI 데모 영상
- **톤앤매너**: 친근하고 전문적

---

## 씬별 상세 스토리보드

### 씬 1: 오프닝 (0:00-0:10)
**비주얼**:
```
[화면 1]
- 검은 배경
- 맑은소프트 로고 페이드인 (중앙)
- 1초 후 "맑은스페이스" 텍스트 등장 (로고 아래)
- 부드러운 페이드 효과

[화면 2]
- 텍스트 오버레이: "회의실 예약, 이제 더 이상 메신저로 공유하지 마세요!"
- 텍스트 애니메이션: 왼쪽에서 슬라이드 인
```

**음향**: 밝고 모던한 배경음악 시작 (BPM 120-130)

---

### 씬 2: 문제점 제시 (0:10-0:25)
**비주얼**:
```
[화면 1] 메신저 화면 (카카오톡/슬랙)
- 여러 사람의 대화창
- "오늘 오후 2시 회의실 사용 예정"
- "줌 쓸 예정입니다"
- "회의실 예약했어요"
- 메시지들이 겹치고 혼란스러운 모습

[화면 2] 중복 예약 상황
- 두 사람이 같은 시간에 같은 회의실 예약
- 혼란스러운 표정/이모지
- "어? 저도 예약했는데요?"

[화면 3] 예약 현황 찾기 어려움
- 긴 대화 내역 스크롤
- 예약 정보를 찾기 어려워하는 모습
```

**텍스트 오버레이**:
- "언제 어디서 회의실을 사용할지 메신저로만 공유하다 보니..."
- "중복 예약, 시간 충돌, 예약 현황 파악 어려움..."

**음향**: 약간 긴장감 있는 음악, 효과음 (알림음)

---

### 씬 3: 솔루션 소개 (0:25-0:50)
**비주얼**:
```
[화면 1] 맑은스페이스 메인 화면
- 달력 뷰 전체 화면
- 공휴일 빨간색으로 표시
- 예약된 시간대 파란색으로 표시
- 줌인 효과로 달력 강조

[화면 2] 날짜 선택
- 마우스 커서가 날짜 위로 이동
- 클릭 애니메이션
- 예약 모달이 부드럽게 열림

[화면 3] 회의실 선택
- 드롭다운 메뉴 열림
- "소회의실 A", "소회의실 B", "소회의실 C", "대회의실" 옵션
- "소회의실 B" 선택 (하이라이트)

[화면 4] 시간 선택
- 시작 시간: 13시 00분 선택
- 종료 시간: 14시 00분 선택
- 드롭다운 애니메이션

[화면 5] 예약 정보 입력
- 예약자: "주혜리" 입력
- 참석자: "이러닝컨설팅팀" 입력
- 목적: "맑은소프트 미팅" 입력
- "줌 계정도 함께 예약하기" 체크박스 체크

[화면 6] 예약 완료
- "예약하기" 버튼 클릭
- 성공 알림 메시지: "예약이 완료되었습니다!"
- 달력에 예약된 시간대 표시
```

**텍스트 오버레이**:
- "맑은스페이스로 모든 회의실 예약을 한눈에!"
- "직관적인 달력에서 날짜를 선택하고"
- "회의실과 시간을 간편하게 예약하세요"
- "줌 계정도 함께 예약할 수 있어요!"

**음향**: 긍정적인 음악, 클릭 효과음

---

### 씬 4: 주요 기능 소개 (0:50-1:10)
**비주얼**:
```
[화면 1] 실시간 예약 현황
- 달력에서 여러 예약이 표시됨
- 각 예약 시간대가 파란색으로 표시
- 마우스 오버 시 예약 정보 툴팁 표시

[화면 2] 중복 예약 방지
- 이미 예약된 시간대 클릭 시도
- 경고 팝업: "해당 시간에 이미 예약이 있습니다"
- "확인" 버튼 클릭

[화면 3] 내 예약 목록
- "내 예약" 탭 클릭
- 예약 목록 카드들이 표시됨
- 날짜, 시간, 회의실 정보가 명확하게 표시
- 스크롤하여 여러 예약 확인

[화면 4] 공휴일 표시
- 달력으로 돌아가기
- 10월 달력에서 공휴일들이 빨간색으로 표시
- "추석", "한글날", "대체공휴일" 등 라벨 표시
```

**텍스트 오버레이** (각 화면마다):
- "✨ 실시간 예약 현황 확인"
- "✨ 중복 예약 자동 방지"
- "✨ 내 예약 목록 한눈에 보기"
- "✨ 공휴일 자동 표시"

**음향**: 기능별로 다른 효과음, 배경음악 지속

---

### 씬 5: 사용 편의성 강조 (1:10-1:25)
**비주얼**:
```
[화면 1] 모바일 화면으로 전환
- 데스크톱 화면에서 모바일 화면으로 전환 애니메이션
- 아이폰/안드로이드 화면

[화면 2] 모바일에서 예약하기
- 터치로 날짜 선택
- 스와이프로 월 변경
- 예약 모달이 모바일에 최적화된 형태로 표시
- 터치로 회의실, 시간 선택

[화면 3] 모바일에서 예약 취소
- "내 예약" 페이지
- 예약 카드에서 "취소" 버튼 터치
- 확인 팝업: "맑은스페이스의 메시지"
- "확인" 버튼 터치
- 예약 취소 완료
```

**텍스트 오버레이**:
- "모바일에서도 언제 어디서나"
- "간편하게 예약하고 확인하세요"

**음향**: 모바일 터치 효과음, 부드러운 전환음

---

### 씬 6: 마무리 (1:25-1:40)
**비주얼**:
```
[화면 1] 주요 화면 모음
- 4분할 화면:
  - 왼쪽 위: 달력 뷰
  - 오른쪽 위: 예약 모달
  - 왼쪽 아래: 내 예약 목록
  - 오른쪽 아래: 모바일 화면
- 각 화면이 순차적으로 하이라이트

[화면 2] 로고 및 메시지
- 맑은스페이스 로고 (중앙)
- "스마트한 회의실 예약 시스템" 텍스트
- URL: malgn-booking.pages.dev
- QR 코드 (선택사항)

[화면 3] CTA (Call to Action)
- "이제 메신저 대신 맑은스페이스를 사용하세요!"
- 페이드 아웃
```

**텍스트 오버레이**:
- "맑은스페이스"
- "스마트한 회의실 예약 시스템"
- "이제 메신저 대신 맑은스페이스를 사용하세요!"

**음향**: 음악이 클라이맥스로 올라가다가 페이드 아웃

## 색상 가이드
- **주요 색상**: 
  - 보라색 (#4f46e5) - 맑은소프트 브랜드 컬러
  - 빨간색 (#ef4444) - 공휴일, 중요 알림
  - 파란색 (#2563eb) - 예약된 시간대
  - 흰색/회색 - 배경 및 텍스트

## 타이포그래피
- **제목**: 굵은 sans-serif 폰트 (예: Noto Sans KR Bold)
- **본문**: 가독성 좋은 sans-serif 폰트 (예: Noto Sans KR)
- **강조**: 색상과 크기로 강조

## 애니메이션 스타일
- **전환**: 부드러운 페이드, 슬라이드
- **인터랙션**: 클릭 시 하이라이트, 호버 효과
- **속도**: 자연스러운 속도 (너무 빠르지 않게)

## 음향 가이드
- **배경 음악**: 밝고 모던한 일렉트로닉 또는 앰비언트
- **효과음**: 
  - 클릭: 부드러운 "틱" 소리
  - 알림: 친근한 "띵" 소리
  - 성공: 긍정적인 "성공" 소리
- **볼륨**: 배경음악 30%, 효과음 70%
